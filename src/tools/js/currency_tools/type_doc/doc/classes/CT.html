Class CT

JS工具，其中的代码都是无关任何项目的代码(ES6，ESM模块写法)。
注：
1、都是在当时最新的谷歌浏览器PC版上测试。
2、注意默认的初始化操作带来的影响
3、该工具支持在浏览器、Node、Deno等多种支持JS的宿主环境中使用！

  • storageCE: ((f) => Function)

    当Storage发生变化的时候，会触发storage事件
    PS：同一个页面要是既执行了这个事件，又发生了localStorage变化，那么这个事件不会被这个页面所触发，
    但是会被同源下的那些有执行这个事件的其他页面或者iframe所触发。

    Returns

    function，返回一个函数，用于解除给“window”绑定的“storage”事件，如：window.removeEventListener( 'storage', f, false )

      • (f): Function

      • Parameters

        • f: any

          函数，会传入一个event
          event.target 事件目标（DOM树中最顶端的目标）
          event.type 事件的类型
          event.bubbles 事件是否正常起泡
          event.cancelable 事件是否可以取消
          event.key 键值发生改变的键名
          event.oldValue 被改变的键名的旧的键值
          event.newValue 被改变的键名的新的键值
          event.url Storage发生变动的页面的URL
          event.storageArea 发生变动的Storage对象

        Returns Function
