setInterval(argObj, time)

定时器，每“time”秒执行一次指定的函数

注：
如果直接设置howClear为true，则这个定时器永远不会执行；
如果直接设置howClear为false并且直接设置howStart为true,则定时器会永远执行下去
如果直接设置howClear为false并且直接设置howStart为false,则定时器会永远执行otherFun下去

注：funDataObj(存储由fun函数设置的数据), clearDataObj(存储由howClear函数设置的数据), startDataObj(存储由howStart函数设置的数据), otherDataObj(存储由otherFun函数设置的数据)
这4个对象按以上顺序用作howClear、howStart、fun、otherFun这四个函数的参数，
用于存储数据，只能把数据添加到对象里，不能把对象修改成其他数据类型
利用引用数据类型达到传递数据的目的
第一次调用setInterval都是空对象的
Parameters:
Name Type Default Description
argObj JSON对象，配置对象，必须
{
fun: function，所要执行的函数(有4个参数funDataObj, clearDataObj, startDataObj, otherDataObj)，必须

howClear: boolean|function，boolean或一个返回值是boolean类型的函数(有4个参数funDataObj, clearDataObj, startDataObj, otherDataObj)，
什么时候取消并清除掉定时器，默认是一个返回false的函数，可选
注：如果不需要取消并清除掉定时器，就传false

howStart: boolean|function，boolean或一个返回值是boolean类型的函数(有4个参数funDataObj, clearDataObj, startDataObj, otherDataObj)，
什么时候执行指定的函数，默认是一个返回true的函数，可选
注：如果任何时候都要执行指定的函数，直到取消并清除掉定时器，那就传true

otherFun: function，当不需要执行fun函数但又不需要取消并清除掉定时器时，执行的备选函数(有4个参数funDataObj, clearDataObj, startDataObj, otherDataObj)，可选
time 1 number，每次执行指定函数的时间间隔，默认1ms，单位ms，可选

throttleFun(fun, wait, immediate) → {function}

节流函数(规定的时间间隔内只触发一次相应的方法，且执行的是第一次的触发状态)
PS:

Parameters:

immediate true boolean，为true时，则回调func是在连续调用开始时就被调用，默认值true，可选
