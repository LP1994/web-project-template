Class: CT

CT(opt)

JS工具，其中的代码都是无关任何项目的代码(ES6，ESM模块写法)。
注：
1、都是在当时最新的谷歌浏览器PC版上测试。
2、注意默认的初始化操作带来的影响
3、该工具支持在浏览器、Node、Deno等多种支持JS的宿主环境中使用！

new CT(opt)

Parameters:
Name Type Description
opt JSON配置对象，可选
{
isSupportBrowser: 布尔值，如果要将CT工具用于浏览器环境，那么传true可以自动初始化浏览器环境的各种功能；
如果不是用于浏览器环境(如：用于NodeJS)，那么传false就不会初始化跟浏览器相关的操作功能。
可选，默认值true。
