gByClass(classN, rootE) → {HTMLCollection|Array}

根据class(类名字符串)查找节点，返回实时集合(HTMLCollection)。如果找不到匹配项，则返回undefined。
注1：getElementsByClassName可以在任何元素上调用，不仅仅是document。 调用这个方法的元素将作为本次查找的根元素。
Parameters:
Name Type Description
classN 单个class(类名字符串)，也可以是数组，以便批量处理，必需
class(类名字符串)格式'.qq'、'.qq.ss'，请将'.q.www,.xx.cc'改写成['.q.www','.xx.cc']，不能是空字符串、空数组
rootE 指定哪个Element对象(单个)作为本次查找的根元素，默认是document对象，可选
也可以是本次查找的根元素的选择器(单个字符串),如果查找到的是节点List，则会选取第一个Element对象作为本次查找的根元素
如果传入的是节点List、jQuery节点对象，则会选取第一个Element对象作为本次查找的根元素
