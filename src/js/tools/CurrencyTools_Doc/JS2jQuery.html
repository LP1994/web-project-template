animate(elem, initialProps, initialParams)

原生JS封装jQuery的animate()，效果一模一样！
Parameters:
Name Type Description
elem 单个节点选择器(字符串)、单个节点选择器组(字符串)、单个节点对象、单个节点List、单个jQuery节点对象，

initialParams 动画效果的配置对象，可选
如：
{
duration: 300, // 表示动画要多久执行完，单位是ms，默认300ms

easing: 'linear', // 动画缓动，默认为'linear'线性匀速，也可以是'swing'(摇摆)，支持函数，会传入一个number类型的参数progress，但需要返回一个number数值

begin: elements => {}, // 动画开始时执行的函数，会有一个'elements'参数，节点数组([Element])

complete: elements => {}, // 动画完成时执行的函数，会有一个'elements'参数，节点数组([Element])

// 动画正在进行中执行的函数，会有四个函数参数
// elements(节点数组[Element])
// complete(number，当前动画完成的百分比，作为小数值，范围是'0'到'1'，'0'表示动画刚开始，'1'表示动画刚执行结束)
// remaining(number，动画还有多少时间结束，以'ms'为单位，'0'表示动画刚执行结束)
// start(number，回调函数开始执行的绝对时间，以'ms'为单位，如：1538238132659这样的时间戳)
progress: (elements, complete, remaining, start) => {}

注：complete处理函数在多个节点('#div_p > p, input')时无法触发！可以嵌套处理！得到每一个元素后再把该元素进行animate处理
节点的顺序是根据文档中节点位置而定的前后！最前的先执行
