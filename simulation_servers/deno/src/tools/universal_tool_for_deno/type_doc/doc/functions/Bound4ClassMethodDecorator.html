函数 Bound4ClassMethodDecorator

  • Bound4ClassMethodDecorator(
        value: Function,
        context: ClassMethodDecoratorContext<(this: unknown, ...args: any) => any>,
    ): void

  • 方法装饰器，用于确保类的非私有方法在调用时，其内部this指向永远都是类、类实例。
    例子：

    class Person {

      public name: string;

      public constructor( name: string ){
        this.name = name;
      }

      @Bound4ClassMethodDecorator
      public greet(): void{
        console.log(`Hello, my name is ${this.name}.`);
      }

    }

    const g = new Person( '张三' ).greet;

    g(); // "Hello, my name is 张三."

    参数

    • value: Function

      被修饰的类的非私有方法本身。

    • context: ClassMethodDecoratorContext<(this: unknown, ...args: any) => any>

      上下文对象。

    返回 void
