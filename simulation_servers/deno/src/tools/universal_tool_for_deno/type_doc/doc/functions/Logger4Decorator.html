  • 通用的装饰器，日志工具。
    PS：
    1、该装饰器用于修饰类的方法、属性、getter、setter、accessor，其执行顺序有个注意事项：
    最优先执行：静态的属性（不分私有性，谁先定义谁先执行）
    然后是：实例属性（不分私有性，谁先定义谁先执行）
    最后是：按照实际调用类的各个功能，自行执行该装饰器。

    参数

    • config: Partial<{ level: keyof Console; message: string; async: boolean }> = {}
      • level

        Console的某个方法名，默认值是"log"，可选。
        常用的有：
        dir、log、warn、info、error、group、assert...
        更多的见：
        https://developer.mozilla.org/docs/Web/API/console

      • message

        日志信息，默认值是空字符串，可选。

      • async

        所修饰的方法是否是异步方法，默认值是false，可选。
