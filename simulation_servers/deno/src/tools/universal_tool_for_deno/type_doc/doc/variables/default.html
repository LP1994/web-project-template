  • MyConsole: typeof MyConsole

    自定义的Console类，用于在控制台输出带颜色、样式的文字，还集成了“chalk”模块（一个可以输出带颜色等样式的文本）的部分函数，这些都被作为静态方法挂载在这个自定义的Console类。
    PS：
    1、“chalk”模块文档：
    https://deno.land/x/chalk_deno

    2、如果使用的是Windows 10 2004(build 19041)或更高版本的Windows OS，请使用系统自带的Windows终端（Windows Terminal）而不是cmd.exe，不然有些效果出不来。

    3、在Deno中要想能正常使用“chalk”得在启用命令中添加“--color=16m”（真彩色支持，1600 万色）标识，加在入口脚本的后面，如：
    deno run -A --config=./deno.json --check --v8-flags=--max-old-space-size=1024000 --reload --watch-hmr ./src/App.mts --color=16m
    支持的标识还有：--color=256（256色支持）、--color（该标识表示默认启用控制台颜色）、--color=16m（真彩色支持，1600 万色）。

  • Bound4ClassMethodDecorator: ((value: Function, context: ClassMethodDecoratorContext<unknown, ((this: unknown, ...args: any) => any)>) => void)
      • (value, context): void

      • 方法装饰器，用于确保类的非私有方法在调用时，其内部this指向永远都是类、类实例。
        例子：

        class Person {

          public name: string;

          public constructor( name: string ){
            this.name = name;
          }

          @Bound4ClassMethodDecorator
          public greet(): void{
            console.log(`Hello, my name is ${this.name}.`);
          }

        }

        const g = new Person( '张三' ).greet;

        g(); // "Hello, my name is 张三."

        参数

        • value: Function

          被修饰的类的非私有方法本身。

        • context: ClassMethodDecoratorContext<unknown, ((this: unknown, ...args: any) => any)>

          上下文对象。

        返回 void

  • Logger4Decorator: ((config?: Partial<{
        level: keyof Console;
        message: string;
        async: boolean;
    }>) => ((value: any, context: DecoratorContext) => any))
      • (config?): ((value: any, context: DecoratorContext) => any)

      • 通用的装饰器，日志工具。
        PS：
        1、该装饰器用于修饰类的方法、属性、getter、setter、accessor，其执行顺序有个注意事项：
        最优先执行：静态的属性（不分私有性，谁先定义谁先执行）
        然后是：实例属性（不分私有性，谁先定义谁先执行）
        最后是：按照实际调用类的各个功能，自行执行该装饰器。

        参数

        • config: Partial<{
              level: keyof Console;
              message: string;
              async: boolean;
          }> = {}

            • context: DecoratorContext

            返回 any
