Class MyConsole

自定义的Console类，用于在控制台输出带颜色、样式的文字，还集成了“chalk”模块（一个可以输出带颜色等样式的文本）的部分函数，这些都被作为静态方法挂载在这个自定义的Console类。
PS：
1、“chalk”模块文档：
https://deno.land/x/chalk_deno

2、如果使用的是Windows 10 2004(build 19041)或更高版本的Windows OS，请使用系统自带的Windows终端（Windows Terminal）而不是cmd.exe，不然有些效果出不来。

Constructors

constructor

  • new MyConsole(config?: {
        color: null | string | number | number[];
        bgColor: null | string | number | number[];
    }): MyConsole

  • 自定义的Console类，用于在控制台输出带颜色、样式的文字，还集成了“chalk”模块（一个可以输出带颜色等样式的文本）的部分函数，这些都被作为静态方法挂载在这个自定义的Console类。
    PS：
    1、“chalk”模块文档：
    https://deno.land/x/chalk_deno

    2、如果使用的是Windows 10 2004(build 19041)或更高版本的Windows OS，请使用系统自带的Windows终端（Windows Terminal）而不是cmd.exe，不然有些效果出不来。

    Parameters

    • config: {
          color: null | string | number | number[];
          bgColor: null | string | number | number[];
      } = ...

      初始化字体颜色（color）、背景色（bgColor），可选。

      • color: null | string | number | number[]

        初始化字体颜色（color），值类型可以是：number[]（rgb值，如：[255, 136, 0]）、string（hex值，如：'#FF8800'）、number（ansi256值，如：194），默认值null，可选。

      • bgColor: null | string | number | number[]

        初始化背景色（bgColor），值类型可以是：number[]（rgb值，如：[255, 136, 0]）、string（hex值，如：'#FF8800'）、number（ansi256值，如：194），默认值null，可选。

    Returns MyConsole

    • Defined in UniversalToolForNode.esm.mjs:477
